Carousel
========

Carousel is a jQuery plugin that provides a simple, basic carousel functionality for your website. It is in now way meant to be fully functional, but is lightweight and easily adaptable to add your own functionality. The idea is if you want something specific, branch the code and modify it however you like. 

Requirements
------------

jQuery 1.2.6


Supported Browsers
------------------

Supported browsers are:
Safari  3
Firefox 2, 3
IE:     6
