Introduction

Carousel is a jQuery plugin that provides a really simple carousel functionality for your website. It is in no way meant to be fully functional, but is small, lightweight and easily adaptable to add your own needs.

Outta da box you get:

  • Animated Horizontal scrolling
  • Custom start index

So branch the code and hack away!!
