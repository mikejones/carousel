Introduction

Carousel is a very simple jQuery plugin that will modify any list of items into a horizontal scrolling carousel. There are a few configuration options, but it has purposely been kept feature light to provide a easily adaptable plugin that you can tailer to your specific requirements.

Requirements

The dependencies needed to run use it have also kept to a minimum. All you need to do to create a carousel is the following:

  • Include jQuery
  • Include the jquery.carousel plugin
  • Explicitly give the list items a width and set their margin and padding to 0

Once these requirements are met you can call the carousel function (specifying a next and previous button)

$("#calendar").carousel({ nextButton: "#next_week_link", prevButton: "#previous_week_link", index: 1, animateDuration: 1000 });

Configuration

The limited configuration privided is:

Key Description Default Required?
nextButton A jQuery selector - Yes
prevButton A jQuery selector - Yes
index The index of the list to be the starting point 0 no
